# 카카오맵 401 에러 해결 가이드

## 🚨 현재 상황
- **에러**: 401 Unauthorized
- **원인**: localhost:3000이 허용된 도메인에 등록되지 않음
- **임시 해결**: 개발 환경에서 데모 지도 표시

## 🔧 해결 방법

### 1. 카카오 디벨로퍼스 콘솔 접속
https://developers.kakao.com

### 2. 앱 설정 → 플랫폼 설정
기존 설정에 **추가**:

**웹 플랫폼:**
- `http://localhost:3000`
- `http://127.0.0.1:3000`
- `https://your-domain.vercel.app` (배포 후)

### 3. 현재 설정 상태 확인
```
API 키: 6b28d3fe635cd8540e6b76f7ec8833fb
등록된 도메인: 확인 필요
```

## 🚀 배포 시 설정

### Vercel 배포 후:
1. 실제 도메인을 카카오 콘솔에 등록
2. 환경변수로 API 키 관리:

**vercel.json**
```json
{
  "env": {
    "KAKAO_MAP_API_KEY": "6b28d3fe635cd8540e6b76f7ec8833fb"
  }
}
```

**사용 예:**
```javascript
const apiKey = process.env.KAKAO_MAP_API_KEY || '6b28d3fe635cd8540e6b76f7ec8833fb';
```

## 📱 현재 임시 해결
개발 환경 (localhost)에서는 데모 지도가 표시됩니다:
- 실제 지도 대신 예쁜 플레이스홀더
- 클릭하면 카카오맵/구글맵 앱으로 연결
- 배포 시 자동으로 실제 지도로 전환

## ✅ 체크리스트
- [ ] 카카오 콘솔에서 localhost:3000 도메인 추가
- [ ] 배포 후 실제 도메인 추가
- [ ] API 키 환경변수화 (보안)
- [ ] 실제 예식장 좌표로 변경

---

**참고**: 현재는 데모 지도로 정상 작동하며, 배포 시 실제 지도가 표시됩니다.
